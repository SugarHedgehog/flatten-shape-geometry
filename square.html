Square Class Demonstration

This page demonstrates the usage of the Square class from the flattenShapeGeometry package.

1. Basic Square Initialization

const square = new Square(5); // Create a square with side length 5

2. Square Properties

// Access square properties square.perimeter; // Get perimeter square.area; // Get area square.type; // Get shape type

3. Vertices and Coordinates

// Get square vertices const vertices = square.vertices;

4. Square Diagonals

// Get square diagonals const lengthDiagonalAC = square.lengthDiagonalAC; const lengthDiagonalBD = square.lengthDiagonalBD;

5. Square Angles

// Access square angles (in radians) square._angleAInRadians; square._angleBInRadians; square._angleCInRadians; square._angleDInRadians;
