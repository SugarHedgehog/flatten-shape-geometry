Rhombus Class Demonstration

This page demonstrates the usage of the Rhombus class from the flattenShapeGeometry package.

1. Basic Rhombus Initialization

let rhombus = new Rhombus(10, {angle: {angleA: Math.PI/12}});

2. Rhombus Properties

// Access rhombus properties rhombus.perimeter; // Get perimeter rhombus.area; // Get area rhombus.type; // Get shape type

3. Vertices and Coordinates

// Get rhombus vertices const vertices = rhombus.vertices;

4. Rhombus Diagonals

// Get rhombus diagonals const lengthDiagonalAC = rhombus.lengthDiagonalAC; const lengthDiagonalBD = rhombus.lengthDiagonalBD;

5. Rhombus Angles

// Access rhombus angles (in radians) rhombus._angleAInRadians; rhombus._angleBInRadians; rhombus._angleCInRadians; rhombus._angleDInRadians;
