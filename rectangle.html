Rectangle Class Demonstration

This page demonstrates the usage of the Rectangle class from the flattenShapeGeometry package.

1. Basic Rectangle Initialization

const rectangle = new Rectangle({width: 10, height: 5});

2. Rectangle Properties

// Access rectangle properties rectangle.perimeter; // Get perimeter rectangle.area; // Get area rectangle.type; // Get shape type

3. Vertices and Coordinates

// Get rectangle vertices const vertices = rectangle.vertices;

4. Rectangle Diagonals

// Get rectangle diagonals const lengthDiagonalAC = rectangle.lengthDiagonalAC; const lengthDiagonalBD = rectangle.lengthDiagonalBD;

5. Rectangle Angles

// Access rectangle angles (in radians) rectangle._angleAInRadians; rectangle._angleBInRadians; rectangle._angleCInRadians; rectangle._angleDInRadians;
