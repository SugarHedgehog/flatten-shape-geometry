Parallelogram Class Demonstration

This page demonstrates the usage of the Parallelogram class from the flattenShapeGeometry package.

1. Basic Parallelogram Initialization

let parallelogram = new Parallelogram({lengthAB: 10, lengthBC: 5}, {angle: {angleA: Math.PI/4}});

2. Parallelogram Properties

// Access parallelogram properties parallelogram.perimeter; // Get perimeter parallelogram.area; // Get area parallelogram.type; // Get shape type

3. Vertices and Coordinates

// Get parallelogram vertices const vertices = parallelogram.vertices;

4. Parallelogram Diagonals

// Get parallelogram diagonals const lengthDiagonalAC = parallelogram.lengthDiagonalAC; const lengthDiagonalBD = parallelogram.lengthDiagonalBD;

5. Parallelogram Angles

// Access parallelogram angles (in radians) parallelogram._angleAInRadians; parallelogram._angleBInRadians; parallelogram._angleCInRadians; parallelogram._angleDInRadians;
